# Папка для медитационной музыки

Поместите сюда файлы медитационной музыки в форматах:
- MP3
- WAV
- OGG

Рекомендуемые названия файлов:
- meditation1.mp3
- meditation2.mp3
- meditation3.mp3
- ambient1.wav
- nature-sounds.mp3
- breathing-music.mp3

Музыка будет автоматически воспроизводиться во время медитационных сессий.
